What I do

  • Android

    I develop Android applications. This is the platform I like the most.
    I use both Android Studio and Eclipse, ANT and Gradle.
    For my work I prefer Scala rather than Java.

  • iOS

    I create iOS applications. Both professionally and for fun.
    I use Objective-C and RubyMotion, XCode and Terminal, Storyboards and code.
    I like Apple hardware.

  • Windows Phone

    I have experience in creating WindowsPhone applications.
    I used to write Windows 8 desktop and tablet apps too.
    Windows is the best system ever.

  • Frontend

    I love frontend technology and working on client side, even JavaScript (i.e. CoffeeScript).
    My experience contains: HTML5, Jade, CSS3, Sass, AngularJS, ReactJS.
    Tools that I use are brunch.io and NodeJS.

  • Backend and Server

    Working on backend is the most exciting from all of the above.
    I like to play with different languages and frameworks.
    So far I've tried: Scala - Play! and Spray.io, Ruby - RoR and Sinatra, Java - JEE and Struts.
